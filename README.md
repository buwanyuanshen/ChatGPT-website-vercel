## 介绍
2024.3.9最新更新，新增自定义模型功能！
1.支持gpt3.5，gpt4，dalle，tts，embedding,moderation,gpt-4-vision-preview等多种模型！
2.支持连续对话，流式输出，参数记忆，保存历史对话，代码markdown高亮显示，访问密码、主题切换等功能！
